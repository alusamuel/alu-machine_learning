# APIs

This directory contains API exercises for data pipeline tasks. The tasks
cover fetching data from REST APIs, handling response status codes,
following paginated endpoints, combining related API resources, and sorting
aggregated results.

## Files

- `0-passengers.py`: lists Star Wars ships that can hold at least a given
  number of passengers using the SWAPI starships endpoint.
- `1-sentience.py`: lists home planets for sentient Star Wars species using
  the SWAPI species and planet endpoints.
- `2-user_location.py`: prints a GitHub user's location while handling
  not-found and rate-limit responses.
- `3-upcoming.py`: displays details for the next upcoming SpaceX launch.
- `4-rocket_frequency.py`: counts SpaceX launches by rocket and prints them
  sorted by launch frequency.

The `*-main.py` files provide sample usage and basic checks for selected
tasks.
