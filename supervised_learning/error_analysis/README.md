# Error Analysis

This directory contains error analysis exercises for supervised learning.
The tasks cover confusion matrices and class-wise evaluation metrics,
including sensitivity, precision, specificity, and F1 score.

## Files

- `0-create_confusion.py`: creates a confusion matrix from one-hot labels
  and predictions.
- `1-sensitivity.py`: calculates sensitivity for each class.
- `2-precision.py`: calculates precision for each class.
- `3-specificity.py`: calculates specificity for each class.
- `4-f1_score.py`: calculates the F1 score for each class.
- `5-error_handling`: answers questions about bias and variance handling.
- `6-compare_and_contrast`: identifies the error analysis metric that
  balances precision and recall.

The `*-main.py` files provide sample usage and basic checks for each task.
