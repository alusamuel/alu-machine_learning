# Pandas

This directory contains pandas exercises for data pipeline tasks.
The tasks cover creating DataFrames, loading CSV data, selecting and
transforming cryptocurrency market data, concatenating datasets, analyzing
summary statistics, and visualizing daily trends.

## Files

- `0-from_numpy.py`: creates a DataFrame from a NumPy array.
- `1-from_dictionary.py`: creates a DataFrame from a dictionary.
- `2-from_file.py`: loads data from a delimited file.
- `3-rename.py`: renames and converts timestamp data.
- `4-array.py`: extracts selected columns as a NumPy array.
- `5-slice.py`: slices selected columns at hourly intervals.
- `6-flip_switch.py`: sorts rows in reverse chronological order and
  transposes the DataFrame.
- `7-high.py`: sorts data by the highest prices.
- `8-prune.py`: removes rows with missing close prices.
- `9-fill.py`: fills missing market data values.
- `10-index.py`: indexes the DataFrame by timestamp.
- `11-concat.py`: concatenates Bitstamp and Coinbase datasets.
- `12-hierarchy.py`: creates a hierarchical index for exchange data.
- `13-analyze.py`: generates descriptive statistics.
- `14-visualize.py`: cleans, aggregates, and visualizes daily data.

The `*-main.py` files provide sample usage and basic checks for selected
tasks.
